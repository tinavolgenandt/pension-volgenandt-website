    **1. Create scroll animation composable** (`app/composables/useScrollReveal.ts`):
    - Wrap VueUse's `useElementVisibility` from `@vueuse/core`
    - Accept a `MaybeComputedElementRef` target parameter
    - Pass `{ threshold: 0.15 }` to trigger when 15% visible. Note: `useElementVisibility` may not support `once` directly -- check the VueUse API. If `once` is not available, use `useIntersectionObserver` with `{ threshold: 0.15 }` instead and stop observing after first visibility trigger (set a `hasBeenVisible` ref to `true` and call `stop()`)
    - Return `{ isVisible }` reactive ref (stays `true` once triggered, never reverts to `false`)
    - SSR-safe: returns `false` on server

    **2. Create ScrollReveal wrapper component** (`app/components/ui/ScrollReveal.vue`):
    - Props: `delay` (number, default 0, ms delay for staggered children), `direction` ('up' | 'left' | 'right' | 'none', default 'up')
    - Uses `useTemplateRef('reveal')` and `useScrollReveal(target)`
    - Toggles `.scroll-reveal--visible` class based on `isVisible`
    - CSS transitions (NOT keyframe animations): `opacity 700ms ease-out, transform 700ms ease-out`
    - Initial state: `opacity: 0`, `transform: translateY(24px)` for 'up' direction (translateX for left/right)
    - Visible state: `opacity: 1`, `transform: translateY(0) translateX(0)`
    - `transitionDelay` style bound to `delay` prop (for staggered children)
    - `@media (prefers-reduced-motion: reduce)`: set `opacity: 1`, `transform: none`, `transition: none` -- no animation at all
    - See RESEARCH.md Pattern 2 for exact implementation

    **3. Create StarRating component** (`app/components/ui/StarRating.vue`):
    - Props: `rating` (number 1-5), `size` (string, default 'size-5' Tailwind class)
    - Compute `fullStars`, `hasHalfStar`, `emptyStars` from rating
    - Render filled stars (`lucide:star` with `fill-waldhonig-400 text-waldhonig-400`), half star (`lucide:star-half`), empty stars (`lucide:star` with `text-sage-300`)
    - Wrap in `div` with `role="img"` and `aria-label` showing "X von 5 Sternen"
    - See RESEARCH.md Code Examples for exact implementation

    **4. Create ScrollIndicator component** (`app/components/ui/ScrollIndicator.vue`):
    - Props: `visible` (boolean)
    - Emits: `click`
    - Renders a `<button>` with `aria-label="Zum Inhalt scrollen"`
    - Icon: `lucide:chevron-down`, `size-8 text-white`
    - Positioned: `absolute bottom-6 left-1/2 -translate-x-1/2`
    - Opacity transitions: `opacity-100` when visible, `opacity-0 pointer-events-none` when not
    - CSS bounce animation: `@keyframes bounce-gentle` with 6-8px translateY, 2s ease-in-out infinite
    - `@media (prefers-reduced-motion: reduce)`: `animation: none`
    - See RESEARCH.md Code Examples for exact implementation
